# cli

Production-ready Node.js backend app. Generated with `create-tcx-backend`.

## Features
- **Framework**: express
- **Language**: TypeScript
- **Database**: mongodb
- **ORM**: mongoose
- **Auth**: jwt

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Configure environment variables in `.env`.

3. Run in development mode:
   ```bash
   npm run dev
   ```

4. Build and start in production:
   ```bash
   npm run build
   npm start
   ```
